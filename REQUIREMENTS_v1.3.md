# MyIsland v1.3 Roadmap (Dynamic Island Pill for Mac)

## Product Direction
MyIsland is a **Dynamic Island pill for macOS** that feels native on both notch and notch‑less MacBooks. The pill is always visible, context‑aware, and becomes a lightweight control surface for system and agent workflows.

## Key Features (v1.3)
1. **Live Activities for Mac**
- Timers, downloads, AirDrop, calendar events, meeting countdown
- Now Playing expansion with scrub preview

2. **Notchless Mac Enhancements**
- Centered pill with adaptive width and subtle depth shadow
- Smart “float” mode when menu bar is hidden

3. **iPhone Continuity Link**
- Show iPhone call / FaceTime / AirPods battery state
- Handoff indicators for Safari / Notes / Music

4. **OpenClaw Integration (Hot Topic)**
- One‑tap prompt to OpenClaw with input dialog
- Agent status badge + last response preview
- Model switcher (OpenAI / Anthropic / GLM / OpenClaw)

5. **System Safety Indicators**
- Mic / camera usage indicator
- Clipboard change indicator
- VPN / network status

## UX Goals
- Always on top (fullscreen + lock screen)
- Zero‑lag updates (<50ms) for key events
- Minimal visual noise; pill stays calm when idle
