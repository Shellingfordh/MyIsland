# MyIsland v1.5 Roadmap (Dynamic Island Pill for Mac)

## Product Direction
MyIsland is a **Dynamic Island pill for macOS** that feels native on both notch and notch‑less MacBooks. The pill is always visible, context‑aware, and becomes a lightweight control surface for system and agent workflows.

## Key Features (v1.5)
1. **Live Activities for Mac**
- Timers, downloads, AirDrop, calendar events, meeting countdown
- Expanded Now Playing with album art + scrub preview
- Battery + charging animation for Mac/iPhone/AirPods

2. **Notchless Mac UX**
- Adaptive pill width based on context intensity
- Optional “floating” pill when menu bar auto‑hides
- “Quiet Hours” mode for distraction‑free sessions

3. **iPhone Continuity Link**
- Show iPhone call/FaceTime status
- Handoff indicators for Safari/Notes/Music
- Universal Clipboard status + quick paste preview

4. **OpenClaw Integration (Hot Topic)**
- One‑tap prompt to OpenClaw with input dialog
- Agent status badge + last response preview
- Model switcher across OpenAI / Anthropic / GLM / OpenClaw
- Auto‑route prompts based on cost/latency preferences

5. **System Safety Indicators**
- Mic/camera usage indicator
- VPN / network status
- Clipboard change indicator
- Screen sharing detection

6. **Pill Interaction Model**
- Tap to expand, drag to reposition, swipe to dismiss alerts
- Short‑lived “attention cards” for notifications
